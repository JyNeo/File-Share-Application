# File-Share-Application
### This is an android application about file-sharing.
#### File-Share-Application achieves the following main functions：
> 1、image-capturing
> 
> 2、image-clipping
> 
> 3、image-synthesising
> 
> 4、file-sharing
> 
> 5、individual information management
>
